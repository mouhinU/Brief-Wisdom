## 🐛 常见问题排查

### 问题1: 点击按钮没反应
**原因**: 浏览器缓存未清除
**解决**: 
```bash
# 强制刷新
Ctrl + Shift + R (Windows/Linux)
Cmd + Shift + R (Mac)
```

### 问题2: 按钮样式不对
**原因**: CSS文件未加载
**解决**: 
1. 打开浏览器控制台（F12）
2. 查看 Network 标签
3. 刷新页面
4. 检查 `resume-editor.css` 是否成功加载（状态码200）

### 问题3: 提示"AI辅助功能已关闭"
**原因**: 顶部AI开关未开启
**解决**: 找到页面顶部的"AI 辅助润色"开关，确保它是开启状态（蓝色）

### 问题4: 提示"权限不足"
**原因**: 未登录或没有resume:manage权限
**解决**: 
1. 先登录系统
2. 确保当前用户有"简历管理"权限

### 问题5: API请求失败
**检查**:
1. 后端服务是否启动（http://localhost:8090）
2. 打开控制台 Network 标签
3. 查看 `/api/resume/ai/polish` 请求
4. 检查响应状态码和内容

1. 点击"✨ AI"按钮

- 错误提示: 红色背景 "✕ 错误信息"

## 💡 技术细节

### API接口
```
POST /api/resume/ai/polish
Content-Type: application/json

{
  "text": "需要润色的文本",
  "fieldType": "description|background|duty|achievement",
  "context": "上下文信息（公司名/项目名）"
}
```

### 响应格式
```json
{
  "success": true,
  "code": "0",
  "msg": "操作成功",
  "data": {
    "result": "润色后的文本"
  }
}
```

### 前端处理逻辑
1. 解析 `responseJson.data.result`
2. 如果为空则解析 `responseJson.result`（兼容旧版本）
3. 显示对比弹窗
4. 用户选择后更新textarea值
